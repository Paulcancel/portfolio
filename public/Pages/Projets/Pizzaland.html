Rest API : "Pizzaland"

Présentation

Nous avons créé en duo une API permettant la gestion d'une base de données d'une pizzeria, en incluant la gestion d'ingrédients, des users, de pizzas et de commandes. Cette API est basée sur la méthode REST et tourne à l'intérieur d'un serveur tomcat, elle gère une base de données en Postgresql et elle utilise également uniquement des objets json. Un système d'identification est également inclus permettant de sécuriser les opérations visant à changer la base de données.
