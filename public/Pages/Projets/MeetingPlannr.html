Application de gestion de rendez-vous "MeetingPlannr"

Présentation

Nous avons réalisé en duo, une application de gestion de rendez-vous professionnels nommés "MeetingPlannr". L'application permet de créer des rendez-vous, de les modifier et de les supprimer. Les rendez-vous sont stockés dans une base de données postgresql. L'application est réalisée en Spring Boot et utilise des jsp pour l'affichage. l'application est également sécurisée par un système de login utilisant Spring Security.

Compétences inclues

  • Spring Boot

  • jsp/css

  • Git

Savoirs-faire

  • Apprentissage de l'ergonomie pour les fonctionnalités.
